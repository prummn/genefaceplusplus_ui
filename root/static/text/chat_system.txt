1. 参考语音配置：
- 在 io/input/audio/ 目录下放置一个名为 ref.wav 的参考音文件（这是 RVC 克隆的目标音色）。
- 若没有对应字幕，系统会通过 ASR 自动生成字幕文件。
2. 发起对话：
- ASR 阶段：用户点击“开始录音”并说话，llm_asr_pipeline.py 调用接口将语音转为文字。
- LLM 阶段：文字发送给智谱 (GLM-4) 或 Gemini。AI 根据上下文回复一段富有情感、口语化的短句。
3. 实时语音克隆 (RVC)：
- 长文本切分：chat_engine.py 会根据标点符号将 AI 回复切分成多个短句。
- 容器化克隆：系统会循环启动 rvc-app Docker 容器，每个容器处理一个短句，以极高性能完成音色转换。
- 音频拼接：使用 pydub 将多个克隆好的短音频无缝拼接成一段完整的 AI 语音。
4. 数字人实时联动：
- 拼接后的完整语音立即发送至 GeneFace++ 容器。
- 数字人模型根据克隆出的声音进行推理，输出带有目标人物音色、口型完全同步的对话视频。
5. 交互历史：所有对话历史会自动保存在 io/history/ 供后续复盘。
6. 更多细节请参照 README.md 中的完整说明和视频教程。
